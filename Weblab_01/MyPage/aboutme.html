Introduction

For the following HTML weblabs we will be using N++, a free to use notepad program, mainly used for computer programming. Within N++, to use html code, one must use tags. Tags are the language of html, and they completely make up every website.


Basics

In order to code html, you need to use a large range of tags. The <head> tag is used to structure the website with background information that is relatively unseen. <body> is the tag in which all of the information and images of a website go under, including paragraph tags (<p>), link references (<a href =>), and image inputs (<img src =>). In order to end each tag, you must follow up the information with an ending tag. Ending tags most often have a / before the normal tag, for example, ≶/p> is the ending tag to <p> (In context it would look something like this, (<p> This is sample text </p>).


Some Formatting

In order to seperate the paragrpahs of text in html, you need to use a specefic tag. The tag needed is <br/>, this breaks hte two peices of text, and puts the second part a line below. In order to further seperate the text, you can use <hr/>, which adds a line seperating the two peices of text, and makes for a larger gap.
